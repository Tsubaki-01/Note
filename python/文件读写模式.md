### 一、基础文本模式（最常用）
这类模式用于处理文本文件（.txt、.py、.csv等），默认按字符读写，需注意编码（如`encoding='utf-8'`）。

| 模式 | 核心含义 | 文件不存在时 | 对原有内容的影响 | 典型使用场景 |
|------|----------|--------------|------------------|--------------|
| `r`  | 只读（默认模式） | 报错 `FileNotFoundError` | 无（仅读取） | 读取配置文件、日志文件 |
| `w`  | 只写 | 创建新文件 | 清空所有原有内容 | 重新写入全新内容（如生成报告） |
| `a`  | 追加写入 | 创建新文件 | 保留原有内容，新内容写在末尾 | 记录日志、追加数据 |
| `x`  | 独占创建并写入 | 创建新文件 | 无（文件全新） | 确保文件不被覆盖（如生成唯一标识文件） |

#### 代码示例（基础模式）
```python
# 1. r模式：读取文件（文件必须存在）
try:
    with open("test.txt", "r", encoding="utf-8") as f:
        content = f.read()
        print("读取内容：", content)
except FileNotFoundError:
    print("文件不存在！")

# 2. w模式：写入文件（清空原有内容）
with open("test.txt", "w", encoding="utf-8") as f:
    f.write("这是w模式写入的内容\n")  # 原有内容会被清空

# 3. a模式：追加内容（保留原有内容）
with open("test.txt", "a", encoding="utf-8") as f:
    f.write("这是a模式追加的内容\n")  # 加在文件末尾

# 4. x模式：创建新文件（文件已存在则报错）
try:
    with open("new_file.txt", "x", encoding="utf-8") as f:
        f.write("独占创建的文件内容")
except FileExistsError:
    print("文件已存在，无法创建！")
```

### 二、二进制模式（加`b`）
在基础模式后加`b`（如`rb`、`wb`、`ab`），用于处理**二进制文件**（图片、视频、音频、压缩包等），按字节读写，**不需要指定encoding**。

| 模式 | 核心含义 | 适用场景 |
|------|----------|----------|
| `rb` | 二进制只读 | 读取图片、视频文件 |
| `wb` | 二进制只写 | 保存图片、生成二进制文件 |
| `ab` | 二进制追加 | 向二进制文件追加数据（如分段下载文件） |
| `xb` | 二进制独占创建 | 确保二进制文件不被覆盖 |

#### 代码示例（二进制模式）
```python
# 读取图片（二进制模式）
with open("image.jpg", "rb") as f:
    img_data = f.read()  # 读取字节数据

# 保存图片（二进制模式）
with open("copy_image.jpg", "wb") as f:
    f.write(img_data)  # 写入字节数据，复制图片
```

### 三、读写混合模式（加`+`）
在基础模式后加`+`（如`r+`、`w+`、`a+`），表示「同时支持读和写」，但不同模式的读写规则有差异，新手需注意文件指针位置。

| 模式 | 核心规则 | 关键注意点 |
|------|----------|------------|
| `r+` | 读写模式，文件必须存在 | 写入会从文件开头覆盖原有内容，读取前需确保指针位置正确 |
| `w+` | 读写模式，先清空文件 | 可读写，但打开文件时已清空内容，读取前需先移动指针到开头 |
| `a+` | 读写模式，追加写入 | 写入始终在末尾，读取前需用`f.seek(0)`将指针移到开头 |

#### 代码示例（r+模式）
```python
# r+：读写（文件必须存在，写入覆盖开头内容）
with open("test.txt", "r+", encoding="utf-8") as f:
    # 先读取原有内容
    content = f.read()
    print("原有内容：", content)
    
    # 将指针移回开头，写入新内容（覆盖开头）
    f.seek(0)
    f.write("覆盖开头的内容")
```

### 总结
1. **基础选择**：只读用`r`，清空重写写用`w`，追加内容用`a`，确保不覆盖用`x`。
2. **文件类型**：文本文件用基础模式（r/w/a/x），二进制文件加`b`（rb/wb/ab）。
3. **读写混合**：`r+`（文件必须存在）、`w+`（清空后读写）、`a+`（追加+读取，需移动指针）。

核心原则：优先使用`with`语句打开文件（自动关闭文件，避免资源泄漏），根据「是否需要读/写」「是否保留原有内容」选择对应模式。

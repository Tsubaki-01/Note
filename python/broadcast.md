要判断两个张量 $\mathbf{A}$ 和 $\mathbf{B}$ 是否兼容，PyTorch 会从它们的**尾部维度（trailing dimensions）**开始逐一比较它们的形状。

只有满足以下任一条件的维度才是兼容的：

1. **维度大小相等。**
2. **其中一个维度大小为 1。**
3. **其中一个张量在那个维度上缺失（即该张量的维度较少）。**

如果两个张量的维度个数不同，PyTorch 会自动在维度较少的张量的**前面**补充大小为 1 的维度，直到两者维度个数相同，然后再进行比较。

------

### 规则图示及案例分析

#### 案例 1：维度大小相等或其中之一为 1

| **维度** | **张量 A 形状** | **张量 B 形状** | **兼容性**   | **结果形状**  | **广播动作**          |
| -------- | --------------- | --------------- | ------------ | ------------- | --------------------- |
| 1        | 4               | 4               | **相等**     | 4             | 无                    |
| 2        | 3               | 1               | **一个为 1** | 3             | B 沿着该维度复制 3 次 |
| 3        | 5               | 5               | **相等**     | 5             | 无                    |
| **结果** | (4, 3, 5)       | (4, 1, 5)       | **兼容**     | **(4, 3, 5)** |                       |

#### 案例 2：维度个数不同（隐式补充 1）

假设 $\mathbf{A}$ 的形状是 $(5, 3)$， $\mathbf{B}$ 的形状是 $(3)$。

1. 首先，PyTorch 将 $\mathbf{B}$ 自动重塑为 $(1, 3)$，使其与 $\mathbf{A}$ 的维度个数相同。
2. 然后，从尾部开始比较：

| **维度** | **张量 A 形状** | **隐式 B 形状** | **兼容性**   | **结果形状** | **广播动作**          |
| -------- | --------------- | --------------- | ------------ | ------------ | --------------------- |
| 1        | 5               | 1               | **一个为 1** | 5            | B 沿着该维度复制 5 次 |
| 2        | 3               | 3               | **相等**     | 3            | 无                    |
| **结果** | (5, 3)          | (1, 3)          | **兼容**     | **(5, 3)**   |                       |

#### 案例 3：与位置编码代码的对应

我们回顾一下刚才的计算： `pos / decay_rate`

- `pos` 形状: $(max\_len, 1)$
- `decay_rate` 形状: $(d_{model} / 2)$

1. 首先，PyTorch 将 `decay_rate` 自动重塑为 $(1, d_{model} / 2)$。
2. 从尾部开始比较：

| **维度** | **pos 形状**    | **隐式 decay_rate 形状** | **兼容性**   | **结果形状**                    | **广播动作**                              |
| -------- | --------------- | ------------------------ | ------------ | ------------------------------- | ----------------------------------------- |
| 1        | $max\_len$      | 1                        | **一个为 1** | $max\_len$                      | `decay_rate` 沿着该维度复制 $max\_len$ 次 |
| 2        | 1               | $d_{model} / 2$          | **一个为 1** | $d_{model} / 2$                 | `pos` 沿着该维度复制 $d_{model} / 2$ 次   |
| **结果** | $(max\_len, 1)$ | $(1, d_{model} / 2)$     | **兼容**     | **$(max\_len, d_{model} / 2)$** |                                           |

#### 案例 4：不兼容的示例

- $\mathbf{A}$ 形状: $(5, 3)$
- $\mathbf{B}$ 形状: $(5, 4)$

| **维度** | **A 形状** | **B 形状** | **兼容性**                     |
| -------- | ---------- | ---------- | ------------------------------ |
| 1        | 5          | 5          | 相等                           |
| 2        | 3          | 4          | **不兼容**（不相等且都不为 1） |
| **结果** |            |            | **不兼容，将报错**             |

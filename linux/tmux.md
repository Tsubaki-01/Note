**Tmux** (Terminal Multiplexer) 的操作主要分为两部分：在终端中运行的**命令行**，以及在 Tmux 内部使用的**快捷键**。

在使用快捷键之前，需要先按下**前缀键 (Prefix)**，Tmux 默认的前缀键是 `Ctrl + b`。也就是说，如果你想使用快捷键 `c`，你需要先按下 `Ctrl + b`，松开后再按 `c`。

以下是 Tmux 最常用的命令和快捷键分类整理：

### 1. 终端命令行 (在普通 Shell 中执行)

这些命令用于在终端中直接管理 Tmux 会话 (Session)。

| **命令**                      | **描述**                          |
| ----------------------------- | --------------------------------- |
| `tmux`                        | 新建一个无名称的会话              |
| `tmux new -s <名称>`          | **新建**一个指定名称的会话 (推荐) |
| `tmux ls`                     | **列出**当前所有的 Tmux 会话      |
| `tmux a` 或 `tmux attach`     | 接入上一次使用的会话              |
| `tmux a -t <名称>`            | **接入**指定的会话                |
| `tmux kill-session -t <名称>` | **关闭/销毁**指定的会话           |
| `tmux kill-server`            | 关闭所有会话和 Tmux 服务          |

------

### 2. 会话 (Session) 快捷键

在 Tmux 内部使用，用于管理和切换整个会话。（需要先按 `Ctrl + b`）

| **快捷键 (Prefix + ...)** | **描述**                                                |
| ------------------------- | ------------------------------------------------------- |
| `d`                       | **断开 (Detach)** 当前会话，将会话放在后台运行 (最常用) |
| `s`                       | 列出所有会话，可通过方向键选择并按回车切换              |
| `$`                       | 重命名当前会话                                          |
| `D`                       | 选择要断开的会话                                        |

------

### 3. 窗口 (Window) 快捷键

一个会话可以包含多个窗口（类似浏览器的标签页）。（需要先按 `Ctrl + b`）

| **快捷键 (Prefix + ...)** | **描述**                           |
| ------------------------- | ---------------------------------- |
| `c`                       | **新建**窗口 (Create)              |
| `p`                       | 切换到**上一个**窗口 (Previous)    |
| `n`                       | 切换到**下一个**窗口 (Next)        |
| `0` 到 `9`                | 切换到指定编号的窗口               |
| `w`                       | 列出所有窗口，可通过方向键选择切换 |
| `,`                       | **重命名**当前窗口                 |
| `&`                       | **关闭**当前窗口 (会有确认提示)    |

------

### 4. 窗格 (Pane) 快捷键

一个窗口可以被分割成多个独立的窗格（分屏）。（需要先按 `Ctrl + b`）

| **快捷键 (Prefix + ...)**  | **描述**                              |
| -------------------------- | ------------------------------------- |
| `%`                        | **左右分屏** (垂直分割)               |
| `"`                        | **上下分屏** (水平分割)               |
| `方向键` (`↑` `↓` `←` `→`) | 在不同的窗格之间移动光标              |
| `x`                        | **关闭**当前窗格 (会有确认提示)       |
| `z`                        | **最大化/恢复**当前窗格 (Zoom)        |
| `{` 或 `}`                 | 与前一个或后一个窗格交换位置          |
| `空格键` (Space)           | 在 Tmux 预设的窗格布局之间切换        |
| `q`                        | 显示窗格编号 (此时按下对应数字可跳转) |
| `Ctrl + 方向键`            | 以步长为 1 调整当前窗格的边缘大小     |

------

### 5. 杂项快捷键

| **快捷键 (Prefix + ...)** | **描述**                                       |
| ------------------------- | ---------------------------------------------- |
| `?`                       | 显示所有快捷键帮助文档 (按 `q` 退出)           |
| `[`                       | 进入复制模式 (可用方向键滚动屏幕，按 `q` 退出) |

**💡 核心工作流提示：**

日常工作中最常见的使用流程是：`tmux new -s work` (创建会话) -> 开始跑任务 -> `Ctrl + b` 然后 `d` (放在后台) -> 下班或换电脑 -> `tmux a -t work` (恢复工作现场)。

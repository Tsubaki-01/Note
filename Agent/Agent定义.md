# AI Agent（人工智能智能体）的定义

## 核心定义
**AI Agent（人工智能智能体）** 是一种能够**感知环境、自主决策并执行动作以实现特定目标的闭环智能系统**。它的本质是从"被动回答问题的工具"进化为"主动完成任务的代理"，核心特征是**自主性**和**目标导向性**。

## 广义与狭义区分
- **广义AI Agent**：人工智能领域诞生超过半个世纪的经典概念，指任何能感知环境、自主决策、采取行动的实体。早期的专家系统、AlphaGo都属于广义Agent范畴。
- **狭义AI Agent（当前行业主流）**：**LLM Agent（大语言模型驱动的智能体）**，2022年ChatGPT爆发后快速崛起。它以大语言模型为核心认知中枢，通过模块化设计补齐LLM的能力短板，实现从"被动文本生成"到"主动完成复杂任务"的跨越。

## 权威定义
1. **李飞飞团队综述定义**：基于大语言模型的自主智能体，是以LLM为核心大脑，整合了感知、记忆、规划、工具调用、行动、反思六大核心能力，能够在开放环境中自主理解用户目标、分解复杂任务、动态调整策略、持续迭代执行，最终完成既定目标的智能系统。

2. **吴恩达的极简定义**：**Agent = LLM + 规划能力 + 工具调用能力 + 记忆能力**，它的核心是让AI从"被动回答问题"变成"主动完成任务"。

3. **Anthropic官方定义**：Agent是LLM能自主决定自己的流程和工具使用的系统，全程掌控任务的完成方式。也就是：它自己决定下一步做什么，自己选择用什么工具，自己判断做完了没有，做完了才停下来。

4. **FIPA（智能体标准化组织）定义**：智能体是"驻留在环境中，能解释从环境获得的数据，并执行对环境产生影响的行动的实体"。

## 四大核心特征
1. **自主性**：无需人工分步干预，能自主理解目标、制定计划、执行动作、处理异常，全程无需人工值守。
2. **目标导向**：所有动作都围绕最终目标展开，能动态调整策略，应对环境变化，确保目标达成。
3. **闭环能力**：形成"**感知→推理→决策→行动→反馈→迭代**"的完整闭环，而非单次、被动的文本响应。
4. **能力可扩展**：通过工具调用、多智能体协同，无限扩展能力边界，彻底突破LLM的固有局限。

## 与传统大模型/聊天机器人的核心区别
| 维度 | 传统大模型/聊天机器人 | AI Agent |
|------|------------------------|----------|
| 交互模式 | 一问一答，被动响应 | 接受目标，自主执行多步任务 |
| 输入输出 | 文本进→文本出 | 目标进→行动+结果出 |
| 任务能力 | 单步、简单任务 | 多步骤、复杂任务，可拆解目标 |
| 工具调用 | 有限或无，需明确指令 | 自主判断何时、如何使用工具 |
| 记忆能力 | 仅短期上下文记忆 | 短期上下文+长期知识库记忆 |
| 错误处理 | 无法自主修正，需用户指出 | 能自我反思、评估并修正错误 |
| 执行逻辑 | 输入→模型→输出 | 感知→思考→行动→反馈→迭代 |

## 核心架构
OpenAI前安全研究VP Lilian Weng提出的经典公式至今仍是行业权威：
**Agent = LLM（核心大脑） + Memory（记忆系统） + Planning（规划推理） + Tool Use（工具调用）**

## 通俗类比
- **LLM** = 一个坐在书房里、智商极高、博览群书的学霸。他能回答你任何书本上的问题，能写文案、解数学题、翻译外语，但他没有手、没有脚、不能出门、记性不好（超过几页纸的内容就会忘）、不知道外面世界的实时变化，只能坐在原地回答你的单次提问。

- **AI Agent** = 给这个学霸配了全套的"行动装备"：给他装上眼睛和耳朵（感知系统）、一个笔记本和档案柜（记忆系统）、一个日程本和规划师（规划系统），还有手机、电脑、银行卡和交通工具（工具系统）。现在你只需要告诉他"帮我安排一次去上海的商务旅行"，他就会自己查机票、订酒店、排行程、约会议，遇到航班延误还会自动调整，最后把完整的行程单交给你。

## On-Policy 与 Off-Policy 的核心区别

在强化学习中，有两个关键策略：行为策略（Behavior Policy）用于与环境交互并生成数据；目标策略（Target Policy）是我们当前真正想要优化和更新的策略。区分两者的唯一标准就是：这两个策略是否是同一个。

| **特性**       | **On-Policy (同策略)**           | **Off-Policy (异策略)**                    |
| -------------- | -------------------------------- | ------------------------------------------ |
| **核心定义**   | 行为策略与目标策略是同一个       | 行为策略与目标策略不同                     |
| **数据利用**   | 必须使用当前模型最新生成的数据   | 可以复用历史数据或人类标注的静态数据       |
| **样本效率**   | 较低（数据更新一次后即作废）     | 较高（数据可存入经验池重复利用）           |
| **算法稳定性** | 较高（直接评估当前行为，偏差小） | 较低（需通过重要性采样等方法处理分布偏移） |
| **典型代表**   | REINFORCE, PPO, GRPO             | Q-Learning, DPO, CISPO                     |
